Embedding Sanity Test Dashboard in Confluence

Option 1: Using the HTML Macro

Once your GitHub repository is set up and the dashboard is deployed to GitHub Pages, you can embed it in Confluence using the HTML macro:

  1. In your Confluence page, click Edit
  2. Place your cursor where you want to embed the dashboard
  3. Type /html to search for the HTML macro
  4. Select the HTML macro from the dropdown
  5. Paste the following code (replace YOUR_USERNAME with your GitHub username):
<iframe 
    src="https://YOUR_USERNAME.github.io/sample-sanity-pipeline/dashboard.html" 
    width="100%" 
    height="800px" 
    frameborder="0"
    allowfullscreen
></iframe>

Option 2: Using the Iframe Macro

If your Confluence instance has the Iframe macro installed:

  1. In your Confluence page, click Edit
  2. Place your cursor where you want to embed the dashboard
  3. Type /iframe to search for the Iframe macro
  4. Select the Iframe macro from the dropdown
  5. In the URL field, enter: https://YOUR_USERNAME.github.io/sample-sanity-pipeline/dashboard.html
  6. Set the width to 100% and height to 800px
  7. Click Insert

For the "Michael Ola" Workspace

Since you mentioned you've created a workspace called "Michael Ola" in Confluence, you can add the dashboard to a page in that workspace using either of the methods above.

Security Considerations

If your Confluence instance is behind a firewall or has Content Security Policy (CSP) restrictions, you may need to:

  • Add the GitHub Pages domain to your allowed domains list
  • Consider using a proxy if direct access to GitHub Pages is not possible
  • Ensure your GitHub repository is public if embedding in a public Confluence page
